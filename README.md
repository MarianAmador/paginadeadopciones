Esta página funciona como un gestor de adopciones. En la página principal se muestran los distintos tipos de animales disponibles para adopción. Al dar clic en el animal que se desea adoptar, el usuario es dirigido a una página correspondiente donde se muestran las imágenes de los animalitos disponibles. Posteriormente, al seleccionar el animal de preferencia, se redirige a un formulario que debe ser llenado para iniciar el proceso de adopción. Finalmente, al enviar el formulario, se muestra una página de agradecimiento que confirma el envío de la solicitud.
